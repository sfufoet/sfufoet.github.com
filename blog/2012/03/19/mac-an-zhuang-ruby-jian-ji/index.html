Mac 安装 Ruby 和 Octopress 简记

Mar 19th, 2012 | Comments

最近学习 Git 这个分布式版本管理系统，这个东东很强大。近几年 GitHub 这个开源社区的后起之秀就是一个基于 Git 的服务。 这几天在看 GotGitHub 这个 github 指南的发现：可以用 github 来折腾静态博客系统：Octopress。

Octopress 的好处看这里：Bye-bye Wordpress. Hello Octopress! - Lucifr

看完之后，准备好折腾的话，从这里开始：

  • Pro Git - Table of Contents
  • Git Community Book 中文版
  • GotGitHub
  • Markdown 工具补完 - 小众软件

然后想尝尝 Octopress 的话，可以参考这三篇安装教程（Octopress 需要 Ruby 环境），主要参考两篇教程（建议交叉参考）：

  • @ghawkgu 推荐的教程：The Pragmatic Studio | Installing Ruby 1.9 and Rails 3 on Mac OS X 注：安装 Ruby 就行，神马 Rail 3 和 SQLite3 就不用理会了。
  • Hello Octopress! - Zespia
  • Octopress安裝筆記 - VCAN

折腾了两天不是这出错就是那出错，搜索 + 问推友，还是安装到最后一步出错。本来宣布放弃，隔了两天，又不死心。继续折腾。终于成功折腾好 Ruby。在这里简单记录一下出错的要点：

  1. 如果你没有安装 Xcode，或者安装了 Xcode 4.3 那么一定要手工安装 GCC
  2. @seansay 提醒：记得在 /usr 啥的目录下要加 sudo
  3. 如果安装 bundle install 的时候提示 Compilation of fsevent_watch failed (see README) (RuntimeError)，这时候只需要要用 bundle update 命令就行了。

Octopress 的好处：用 Markdown 来写文章真是好方便呀，看起来不费神，写起来也不费力。这文章的永久地址居然自动拼音化，内牛满面啊。

Octopress 这货的缺点就是写文章需要用命令行，更新也需要命令行，学习成本还是挺高的，要会点 git，了解 github，会 Markdown。

正如 Octopress 官方副标题一样：A blogging framework for hackers，这货真的是非常小众非常 hacker。
